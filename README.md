# Numbers
by [daikikojima](https://github.com/daikikojima)

## About...
NumbersはDNNを用いた数字認識アプリです.
[こちら](https://numberpredict.herokuapp.com/)に完成版があります.
今回, HerokuにDeployする, ということで, cpuを用いて容易に計算できるネットワークにしております.

## 環境
* python: 3.6系
* flask: 1系
* chainer: 4系

としております.

## 実行方法
```bash
pip install -r requirements.txt
python app.py
```
